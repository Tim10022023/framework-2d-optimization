# 🧠 Framework zur Optimierung von zweidimensionalen Funktionen

Dieses Projekt ist Teil einer Studienarbeit und hat das Ziel, ein **dockerbasiertes Web-Framework** zu entwickeln,  
das die **Visualisierung und Optimierung von zweidimensionalen Funktionen** ermöglicht.

Über eine **Weboberfläche** können Nutzer Punkte in einem **2D-Koordinatensystem** auswählen und erhalten den jeweiligen Funktionswert.  
Ziel ist es, durch geschickte Auswahl Strategien zur **Suche nach Hoch- oder Tiefpunkten** zu entwickeln.

Dozierende können Sessions starten, Funktionen oder Karten konfigurieren und sehen die Ergebnisse der Teilnehmenden in einer Rangliste.  
Später soll das System auch **Optimierungsalgorithmen als virtuelle Spieler** unterstützen, um diese visuell vergleichen zu können.

**Technologien:**

- Python (FastAPI)
- React (Vite)
- Docker
- SQLite / PostgreSQL

**Einsatz:**  
Lehrtool zur Veranschaulichung von Optimierungsverfahren in der Vorlesung.
