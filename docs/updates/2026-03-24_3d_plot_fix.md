# Update: 2026-03-24 - 3D Plot Stability and UI Optimization

## Bug Fixed: 3D Surface Plot Stability
The 3D surface plot in the Teacher View was unresponsive when using the step slider and would reset its view periodically (every 3 seconds) due to background polling.

### Optimization Highlights
- **Layout Stabilization**: Plotly layout objects in `FunctionSurfacePlot` and `FunctionContourPlot` are now stable as long as the plot visibility doesn't change. This prevents Plotly from resetting the camera state on every slider tick.
- **Polling Isolation**: In `TeacherInspectPanel`, the participant path (`points`) is now memoized based on stable scalar values (`participant_id` and `clicks.length`). 
- **Reference Preservation**: Direct passing of the `points` array to sub-components avoids reference breakage that previously occurred with in-line `.map()` calls.

### Result
- Smooth slider interaction while simultaneously rotating the 3D plot.
- No more "flashing" or resetting of the 3D view during the 3-second polling intervals.
